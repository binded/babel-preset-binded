# babel-preset-binded

Babel preset for Binded Node.js packages. Based on [eslatest-node6](https://github.com/binded/babel-preset-eslatest-node6) with object rest/spread enabled.

## Install

```bash
npm install --save-dev babel-preset-binded
```

## Usage

**.babelrc**

```json
{
  "presets": ["binded"]
}
```
